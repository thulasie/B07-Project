# Smart Air

Group project repository for UTSC 2025 Fall CSCB07 course
This repository contains source code for an Android app named SMART AIR

Smart Air's purpose is to help parents, healthcare providers, and childrens with asthma(patient) to communicate better

To install the app on your device:
git clone https://github.com/thulasie/B07-Project
on the path of your choice

Some usages of Smart Air can are:
* Sharing information with healthcare providers
* Quick view of each children's summary
* Easy recording of patients' conditions
* Realtime update on patients' health
